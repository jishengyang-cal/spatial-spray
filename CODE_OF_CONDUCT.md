# Code of Conduct

Spatial Spray is an AR creation and discovery scaffold. Discussions should stay technical, evidence-based, and respectful.

## Expected behavior

- Keep feedback focused on product rules, AR boundaries, privacy, moderation, Apple workflow, code, tests, and docs.
- Be clear when a capability requires macOS, Xcode, ARKit, RealityKit, signing assets, simulator access, or real device testing.
- Treat location, identity, user-generated content, and moderation concerns as first-class product risks.
- Redact credentials, tokens, device identifiers, signing material, location-sensitive data, and private infrastructure details.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing credentials, certificates, provisioning profiles, private keys, user data, precise private locations, or production host details.
- Encouraging unsafe bypasses of provider login, privacy, moderation, Apple signing, approval, or remote access boundaries.
- Repeated off-topic disruption of technical discussions.

## Enforcement

Maintainers may edit, hide, close, or lock issues, pull requests, discussions, or comments that violate this code of conduct.
